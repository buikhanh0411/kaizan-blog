# Công nghệ mình dùng

Đây là danh sách các công cụ, phần mềm và phần cứng mình sử dụng hằng ngày để làm việc, lập trình và giải trí. Cảm hứng từ phong cách [uses.tech](https://uses.tech/).

## Phần cứng
- **Máy tính chính:** Hiện tại mình đang dùng một chiếc máy tính chạy Linux (sau khi chiếc Chuwi Minibook thân yêu đã "nghỉ hưu").
- **Bàn phím:** Một chiếc bàn phím cơ nhỏ gọn để gõ code cho sướng tay.

## Phần mềm & Lập trình
- **Hệ điều hành:** Linux (Dùng Linux mang lại cảm giác bình yên và kiểm soát hoàn toàn hệ thống).
- **Trình soạn thảo:** Neovim (nvim), VS Code (code), Vim & Nano.
- **Terminal:** Alacritty hoặc Kitty.
- **Containerization:** [Docker](https://www.docker.com/) (Docker & dockerd).
- **Quản lý mã nguồn:** Git, [GitHub CLI](https://cli.github.com/) (gh) và **Lazygit** (công cụ TUI cực tiện lợi).

## Ngôn ngữ & Build Tools
- **Ngôn ngữ:** 
  - **Python:** Phiên bản 3.10 & 3.12, kèm theo **bpython** (interpreter có giao diện đẹp).
  - **JavaScript/TypeScript:** Sử dụng với **Bun** (Bun & bunx) - cực nhanh và tiện lợi.
  - **C/C++:** Bộ biên dịch **GCC/G++** (phiên bản 11 & 12).
- **Build Tools:** Make và CMake.
- **Automation:** **Playwright** (Dùng để automation testing và web scraping).

## Server & Tự vận hành
- **Triển khai:** [Cloudflare Tunnel](https://www.cloudflare.com/products/tunnel/) (Giúp public server nội bộ ra internet một cách an toàn).
- **Hosting:** Tự host trên server cá nhân (Home server).

## Trang web này
- **SSG:** [Eleventy](https://www.11ty.dev/) (Đơn giản, hiệu quả và cực kỳ nhẹ).
- **Styling:** [Simple.css](https://simplecss.org/) (Tối giản là trên hết).

## Khác
- **Trình duyệt:** Firefox (Chính), Google Chrome (Dùng để debug web).
- **Ghi chú:** Đơn giản là các file **.txt** :)))
